# Release Notes — v1.0.0 Reference Candidate

## Summary

This release candidate turns the repository into a structured cybersecurity
engineering reference platform. It includes a curated published index, draft
artifact libraries, validation scripts, architecture diagrams, and model
reference-grade pages.

## Highlights

- Published index added.
- Detection rule metadata library added.
- CVE reference library added.
- Incident response playbook library added.
- Synthetic IOC library added.
- Architecture diagrams added.
- Validation automation added.

## Known Limitations

- Draft CVE pages must be reviewed against current vendor guidance before being
  promoted to published.
- Detection rules are platform-neutral metadata, not SIEM-specific production
  queries.
- IOC reports are synthetic examples unless explicitly stated otherwise.
- Diagrams are Mermaid-based and should be reviewed visually in GitHub.
