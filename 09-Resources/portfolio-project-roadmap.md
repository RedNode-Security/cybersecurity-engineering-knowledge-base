# Portfolio Project Roadmap

## Overview

This roadmap turns the knowledge base into visible portfolio projects. Each
project should produce documentation, sample data, safe code, and a clear story
about security engineering value.

## Project 1 — Detection Engineering Pack

Deliverables:

- Detection metadata standard
- Sample detection JSON
- Sample Windows authentication events
- Detection testing strategy
- SOC triage workflow

Portfolio story:

```text
I built a detection engineering workflow that converts threat hypotheses into documented, testable detections with triage and response guidance.
```

## Project 2 — IOC Lifecycle Tool

Deliverables:

- IOC schema
- Sample IOC report
- Local enrichment design
- Expiration report
- Markdown output

Portfolio story:

```text
I built a safe local workflow for validating and operationalizing indicators with confidence and expiration handling.
```

## Project 3 — Cloud Identity Detection

Deliverables:

- AWS IAM hardening checklist
- CloudTrail detection examples
- AWS IR playbook
- Azure identity detection guide
- Synthetic cloud audit events

Portfolio story:

```text
I mapped cloud identity risks to audit logs, detections, triage steps, and response actions.
```

## Project 4 — AI Security Threat Model

Deliverables:

- Prompt injection threat model
- RAG checklist
- Agent tool-use control model
- Evaluation examples
- Sample evaluation JSON

Portfolio story:

```text
I designed controls for LLM, RAG, and agent systems that separate trusted instructions from untrusted data and govern tool use.
```

## Project 5 — Security Architecture System

Deliverables:

- SOC reference architecture
- XDR architecture
- Zero Trust architecture
- Autonomous SOC model
- Executive metrics model

Portfolio story:

```text
I designed a full security operating model that connects telemetry, detections, response, automation, metrics, and governance.
```
