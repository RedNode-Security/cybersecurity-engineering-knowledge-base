# Incident Response

## Overview

Incident response converts security alerts and evidence into containment, eradication, recovery, and lessons learned.

## Standard Flow

1. Preparation
2. Identification
3. Triage
4. Containment
5. Eradication
6. Recovery
7. Lessons learned
